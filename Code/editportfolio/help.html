Como editar o texto?

  1. Seleciona a opção Texto.
  2. Para editar o texto, digite o texto que deseja inserir na caixa de texto..
  3. Para editar o tamanho da fonte, selecione o tamanho desejado no primeiro dropdown.
  4. Para editar a decoração do texto, selecione a decoração desejada no segundo dropdown.
  5. Para editar o peso da fonte, selecione a decoração desejada no terceiro dropdown.
  6. Desça a página e selecione a opção "Aplicar".
  7. Selecione no objeto que deseja aplicar as edições. OBS: Serão aplicadas todas as configurações selecionadas que estiverem disponíveis naquele item ao objeto selecionado.

Como editar a cor?

  1. Seleciona a opção Cor.
  2. Para editar a cor, selecione o ColorPicker e escolha a cor desejada.
  3. Desça a página e selecione a opção "Aplicar"
  4. Selecione no objeto que deseja aplicar as edições. OBS: Serão aplicadas todas as configurações selecionadas que estiverem disponíveis naquele item ao objeto selecionado.

Como editar as decorações?

  1. Seleciona a opção Decor.
  2. Para editar a opacidade, selecione o número de 0 a 1 desejado ao deslizar o slider.
  3. Para editar a borda, digite um comando entre as seguintes opções:
    • none - nenhuma borda.
    • hidden - borda oculta.
    • dotted - borda pontilhada com círculos.
    • dashed - borda pontilhada com retas.
    • solid - borda sólida.
    • double - borda dupla.
    • groove - borda 3D.
    • ridge - borda 3D, mais discreta.
    • inset - borda 3D, com efeito de posição interna.
    • outset - borda 3D, com efeito de posição externa.
  4. Para editar o padding, digite um número de acordo com o tamanho desejado.
  5. Desça a página e selecione a opção "Aplicar".
  6. Selecione no objeto que deseja aplicar as edições. OBS: Serão aplicadas todas as configurações selecionadas que estiverem disponíveis naquele item ao objeto selecionado.
